# 🗂️ User Story Mapping — Clarity

> *O mapa de histórias organiza o que o usuário faz na ordem que ele faz — e deixa claro o que entra no MVP sem perder a visão do todo.*

---

## Jornada principal: Ana, a profissional em transição

A jornada cobre desde o momento em que Ana ouve falar do Clarity até ela concluir seu primeiro objetivo de carreira.

---

## Mapa Completo

### 🧭 Atividade 1 — Descobrir e criar conta

| Nível | História |
|-------|---------|
| **Atividade** | Entrar no Clarity pela primeira vez |
| Tarefa | Acessar a landing page e entender o que o produto faz |
| Tarefa | Criar conta com e-mail ou Google |
| Tarefa | Receber e-mail de boas-vindas |
| — | **← LINHA DE CORTE MVP →** |
| V2 | Login com LinkedIn |
| V2 | Onboarding social (ver o que outros usuários fizeram) |

### 🎯 Atividade 2 — Definir objetivo de carreira

| Nível | História |
|-------|---------|
| **Atividade** | Dizer ao Clarity aonde quer chegar |
| Tarefa | Responder: "Qual é o seu objetivo profissional?" |
| Tarefa | Definir prazo (6 meses, 12 meses, 2 anos) |
| Tarefa | Confirmar área de atuação atual |
| — | **← LINHA DE CORTE MVP →** |
| V2 | Receber sugestão de objetivo com base no perfil |
| V2 | Compartilhar objetivo com mentor ou gestor |

### 📚 Atividade 3 — Registrar o que já aprendeu

| Nível | História |
|-------|---------|
| **Atividade** | Centralizar o aprendizado no Clarity |

| Tarefa | Categorizar por área (produto, dados, liderança...) |
| Tarefa | Ver cursos organizados no perfil |

### 📊 Atividade 4 — Acompanhar evolução

| Nível | História |
|-------|---------|

| Tarefa | Identificar lacunas em relação ao objetivo |

### ✅ Atividade 5 — Fazer check-in semanal

| Nível | História |
|-------|---------|
| **Atividade** | Manter consistência semana a semana |
| Tarefa | Receber notificação toda segunda-feira |

| Nível | História |
|-------|---------|
| **Atividade** | Chegar onde queria chegar |
| Tarefa | Marcar objetivo como concluído |
| Tarefa | Ver resumo da jornada (tempo, cursos, check-ins) |
| Tarefa | Definir próximo objetivo |
| — | **← LINHA DE CORTE MVP →** |
| V2 | Exportar portfólio automático com a jornada completa |

---

## O que entra no MVP

Com a linha de corte aplicada:

1. ✅ Criar conta e onboarding
2. ✅ Definir objetivo com prazo
3. ✅ Registrar cursos manualmente
4. ✅ Dashboard de evolução básico
5. ✅ Check-in semanal com notificação
6. ✅ Trilha personalizada (versão simples, sem IA)
7. ✅ Conclusão de objetivo

---

## Lição aprendida ao montar este mapa

> Antes do user story mapping, a equipe queria começar pela feature de portfólio automático — porque parecia mais impressionante para demos. O mapa deixou claro que o portfólio só faz sentido depois que o usuário completou pelo menos um objetivo. Sem dados de jornada, o portfólio é vazio. A sequência importa.
