# 10-prototipo 🎯

Esta pasta contém o protótipo clicável e navegável do **Clarity**, projetado especificamente para demonstrar a proposta de valor e a jornada do usuário no MVP.

## 🚀 Como visualizar o Protótipo

1. Abra a pasta `10-prototipo` no seu computador.
2. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente em qualquer navegador web (Chrome, Edge, Safari, Firefox).
3. **Para colocar no ar no seu GitHub:** Quando você fizer o push deste projeto para o seu repositório público, você pode ativar o **GitHub Pages** nas configurações do repositório apontando para a raiz ou para esta pasta. Assim, qualquer recrutador poderá clicar no link direto e interagir com o app!

## 📱 Funcionalidades Core Demonstradas (Mapeadas do MVP/PRD):
- **Fluxo de Onboarding Completo:** Definição automatizada do objetivo de carreira, prazos (6 meses, 12 meses, 2 anos) e validação da área de origem (ex: Analista de QA).
- **Dashboard Dinâmico:** Visualização do objetivo fixado no topo de todas as telas, barra de progresso em tempo real e contadores de engajamento.
- **Registro de Cursos (Sem Fricção):** Simulação do cadastro de novos certificados alimentando o repositório central.
- **Trilha de Competências Interativa:** Divisão exata sugerida no PRD (Habilidades Técnicas, Comportamentais e Conhecimento de Mercado) com cálculo em tempo real da barra de evolução.
- **Check-in Semanal com Loop Emocional:** Formulário quantitativo e seletor qualitativo de sentimentos com tela de celebração e reforço positivo pós-envio.
- **Encerramento de Ciclos (Activity 7):** Botão para simular a conclusão do objetivo estratégico e exportação de portfólio.
