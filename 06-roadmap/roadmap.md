# 🗺️ Roadmap — Clarity 2026

> *Um roadmap é uma aposta estratégica, não uma promessa de entrega. Ele muda à medida que aprendemos.*

---

## Filosofia do roadmap

Organizamos por **temas estratégicos**, não por lista de features. Cada trimestre tem uma intenção clara — e as features servem a essa intenção, não o contrário.

---

## Roadmap 2026

| Trimestre | Tema | Objetivo |
|-----------|------|----------|
| **Q1** | Fundação | MVP sólido, primeiros 1.000 usuários reais |
| **Q2** | Engajamento | Criar o hábito, aumentar retenção |
| **Q3** | Inteligência | Personalização que impressiona |
| **Q4** | Expansão | Escala e novos mercados |

---

## Q1 — Fundação (Jan–Mar)

**Meta:** Lançar MVP. Atingir 1.000 usuários ativos. NPS > 35.

| Feature | Prioridade | Resultado esperado |
|---------|-----------|-------------------|
| Onboarding com objetivo de carreira | 🔴 Must | Usuário entende o produto no primeiro uso |
| Registro de cursos e certificados | 🔴 Must | Base de dados do usuário criada |
| Dashboard de evolução | 🔴 Must | Usuário vê valor imediatamente |
| Trilha personalizada (básica) | 🔴 Must | Direção clara após onboarding |
| Check-in semanal | 🟡 Should | Engajamento recorrente desde o início |

**KPIs do trimestre:**
- 1.000 usuários ativos
- Onboarding completion rate > 70%
- D7 retention > 30%

---

## Q2 — Engajamento (Abr–Jun)

**Meta:** Criar hábito. Churn mensal < 5%. MAU de 4.000.

| Feature | Prioridade | Resultado esperado |
|---------|-----------|-------------------|
| Histórico de check-ins e evolução | 🔴 Must | Usuário vê progresso ao longo do tempo |
| Metas com prazo e marcos | 🔴 Must | Comprometimento e senso de conquista |
| Notificações inteligentes | 🟡 Should | Engajamento passivo sem ser invasivo |
| Exportar portfólio (v1) | 🟡 Should | Evidência de valor compartilhável |
| Plano premium | 🟡 Should | Primeira fonte de receita |

**KPIs do trimestre:**
- MAU: 4.000
- Churn mensal < 5%
- Conversão free → pago > 8%

---

## Q3 — Inteligência (Jul–Set)

**Meta:** Personalização que surpreende. MAU de 7.000.

| Feature | Prioridade | Resultado esperado |
|---------|-----------|-------------------|
| IA para recomendações de trilha | 🔴 Must | Sugestões que fazem sentido para o objetivo |
| Portfólio automático compartilhável | 🔴 Must | Viral loop — usuário compartilha, atrai outros |
| Insights semanais personalizados | 🟡 Should | "Você estudou X horas este mês — 20% a mais que o mês passado" |
| Integração com LinkedIn (básica) | 🟡 Should | Importar experiência profissional |

**KPIs do trimestre:**
- MAU: 7.000
- NPS > 50
- Portfólios compartilhados: > 500/mês

---

## Q4 — Expansão (Out–Dez)

**Meta:** 10.000 MAU. Explorar B2B.

| Feature | Prioridade | Resultado esperado |
|---------|-----------|-------------------|
| Painel para empresas (B2B pilot) | 🔴 Must | Segunda linha de receita |
| Networking entre usuários com objetivos similares | 🟡 Should | Engajamento comunitário |
| App mobile nativo (iOS + Android) | 🟡 Should | Acessibilidade e uso em qualquer lugar |
| API para plataformas de cursos | 🟢 Could | Importação automática de certificados |

**KPIs do trimestre:**
- MAU: 10.000
- ARR projetado para 2027 definido
- Pilot B2B: 3 empresas

---

## O que NÃO está no roadmap (e por quê)

| Ideia | Motivo de exclusão |
|-------|-------------------|
| Marketplace de cursos | Não é o negócio do Clarity — somos a camada de estratégia |
| Gamificação pesada | Pesquisa mostrou que parece infantil para o público |
| Comunidade tipo Reddit | Alto custo de moderação, baixo fit com o produto |
| Coaching humano integrado | Escala complicada; pode ser parceria futura |
