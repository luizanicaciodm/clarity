# 👥 Personas — Clarity

> *Uma persona não é uma invenção. É uma síntese das pessoas reais que entrevistamos.*

---

## Como chegamos aqui

Depois de 8 entrevistas, padrões começaram a aparecer. Agrupamos os perfis em duas personas primárias e uma secundária, não como personagens fictícios, mas como representações reais de quem vai usar o Clarity todo dia.

---

## Persona 1 — Ana, a Profissional em Transição

```
┌──────────────────────────────────────────────────────────────┐
│  ANA                                                         │
│  28 anos · QA → Product Manager                              │
│  São Paulo · Salário: R$ 5.500/mês                           │
│  Estuda: à noite e nos fins de semana                        │
└──────────────────────────────────────────────────────────────┘
```

### Contexto

Ana é QA há 3 anos. Gosta da área de tecnologia, mas sente que quer estar mais perto das decisões de produto. Já fez um curso de PM, está estudando métricas, assiste podcasts de produto, mas quando alguém pergunta "você está pronta para virar PM?", ela trava. Não sabe como responder com confiança.

### Objetivos

- Conseguir sua primeira vaga como Product Analyst ou PM Jr
- Entender o que ainda falta no seu perfil
- Montar um portfólio que mostre evolução real

### Frustrações

- Faz cursos mas não sabe se está estudando as coisas certas
- Não consegue organizar o que já aprendeu de forma clara
- Sente que está "dando voltas" sem chegar a lugar nenhum
- Tem medo de aplicar para vagas e ser rejeitada por falta de preparo

### Comportamento

| Item | Detalhe |
|------|---------|
| Ferramentas que usa | Notion (bagunçado), LinkedIn, Udemy |
| Frequência de estudo | 3–4x por semana |
| Como decide o que estudar | Vagas que vê no LinkedIn |
| Maior bloqueio | Não saber se está evoluindo |

### Frase que define

> *"Eu estudo muito. Só não sei se estou estudando o que precisa."*

## Persona 2 — Bruno, o Analista que Quer Crescer

```
┌──────────────────────────────────────────────────────────────┐
│  BRUNO                                                       │
│  31 anos · Analista de Dados Pleno                           │
│  Curitiba · Salário: R$ 7.000/mês                            │
│  Quer: promoção para Sênior em 12 meses                      │
└──────────────────────────────────────────────────────────────┘
```

### Contexto

Bruno é bom no que faz. Recebe elogios do time, entrega no prazo, nunca deu problema. Mas olha para o Sênior do lado e não entende exatamente o que precisa fazer para chegar lá. Já tentou conversar com o gestor, que disse "continue evoluindo". Precisa de mais do que isso.

### Objetivos

- Conseguir promoção para Analista Sênior
- Ter clareza do que falta no seu desenvolvimento
- Mostrar para o gestor que está evoluindo com dados

### Frustrações

- Não tem visibilidade clara do que precisa para ser promovido
- Faz cursos por intuição, não por estratégia
- Sente que a empresa não investe no seu desenvolvimento
- Não sabe como evidenciar evolução em uma conversa de carreira

### Comportamento

| Item | Detalhe |
|------|---------|
| Ferramentas que usa | Excel, planilhas, LinkedIn Learning |
| Frequência de estudo | 1–2x por semana |
| Como decide o que estudar | Cursos que a empresa paga |
| Maior bloqueio | Falta de estratégia e direção |

### Frase que define

> *"Eu quero crescer. Só não sei exatamente o que 'crescer' significa para mim agora."*

## Persona 3 (Secundária) — Carla, a Recém-Formada

```
┌──────────────────────────────────────────────────────────────┐
│  CARLA                                                       │
│  24 anos · Recém-formada em Administração                    │
│  Belo Horizonte · Primeiro emprego na área de tech           │
│  Estudou: tudo que apareceu, sem filtro                      │
└──────────────────────────────────────────────────────────────┘
```

### Contexto

Carla se formou animada, fez vários cursos durante a graduação — Python, Excel, UX, produto, mas agora não sabe o que fazer com tudo isso. Tem vontade, tem energia, mas não tem direção. Aplica para vagas diferentes toda semana e não recebe retorno.

### Frase que define

> *"Aprendi muita coisa. Só não sei ainda o que quero ser."*
